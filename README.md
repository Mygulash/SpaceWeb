SpaceWeb is a website that shows you the Astronomy Picutre of the Day from NASA
When you enter the website it load the todays picture but there is a date picker in which you can set the date and the picture for that date will come up.

It fetches the NASA APIs and it gets:
- Title
- Image (or video)
- Description
All of above will be shown to you. 

The website is coded using Javascript and designed using HTML and CSS. 
It is hosted via GitHub Pages. 

The main problem is probably the fail rate of the fetching. Connection fails are sadly often.
